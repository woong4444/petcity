메인 배너 등록

메인 화면에 표시할 이미지와 연결 링크를 등록합니다.

목록으로

기본 정보

관리자 목록에서 배너를 구분할 수 있는 정보를 입력합니다.

배너에 표시될 대표 제목입니다. 0 / 200
제목 아래에 표시될 설명입니다. 0 / 500

배너 이미지

이미지 파일을 업로드하거나 이미지 주소를 직접 입력할 수 있습니다.

선택된 파일 없음

JPG, JPEG, PNG, WEBP, GIF, AVIF 이미지 주소를 입력해 주세요.

연결 링크

사용자가 배너를 클릭했을 때 이동할 주소를 입력합니다.

프로젝트 내부 주소와 외부 인터넷 주소를 모두 입력할 수 있습니다.

링크 입력 예시
/hospitals /hospitals/map /boards https://example.com/event

노출 설정

배너의 순서와 노출 기간을 설정합니다.

입력한 순서부터 기존 배너가 뒤로 이동합니다.

입력하지 않으면 등록 후 바로 노출할 수 있습니다.

입력하지 않으면 종료일 없이 계속 노출됩니다.
